# Node.js App Template

A simple template for Node.js applications built on these principles:

## Principles

**Standards over customization** — Use established tools at their defaults. Defer opinions to upstream maintainers rather than accumulating project-specific overrides.

**Security from the ground up** — Security is structural, not advisory. The environment enforces constraints by design rather than relying on convention or documentation.

**Strict with no escape hatches** — Maximum strictness at every level. There is no "fix it later" category.

**Automation over discipline** — Quality gates run automatically. Correct code should be the path of least resistance.

**Reproducible environments** — Every developer works in an identical environment. Consistency is more valuable than convenience.

**Great developer experience** — The environment provisions itself and repetitive tasks are automated. A new developer should be productive immediately.
